Enable PUA always-on mode:

1. Ensure `~/.pua/` directory exists
2. Write `{"always_on": true}` to `~/.pua/config.json`
3. Output confirmation: > [PUA ON] Starting now, every new session will auto-load PUA. No slackers on this team.
